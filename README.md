## 주요 구현 사항
<ul>
  <li>Intersection Observer API를 이용한 무한스크롤</li>
  <li>styled-components를 사용해 스타일링</li>
  <li>react-router-dom을 이용해 페이지 라우팅</li>
  <li>1세대 포켓몬 ReacQuery로 다량 api 요청</li>
</ul>
